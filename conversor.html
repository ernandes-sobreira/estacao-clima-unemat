Conversor HOBOlink → daily.json

Estação Climatológica UNEMAT · Cáceres-MT · Pantanal Norte
Como usar esta ferramenta
  1. Exporte do HOBOlink o arquivo .xlsx do período desejado (dados de 5 em 5 minutos).
  2. Na Etapa 1, carregue o daily.json atual do repositório (baixe de: github.com/ernandes-sobreira/estacao-clima-unemat → arquivo daily.json → botão Raw → salvar como).
  3. Na Etapa 2, arraste o arquivo .xlsx do HOBOlink.
  4. Confira os dias novos que serão adicionados.
  5. Na Etapa 3, baixe o novo daily.json atualizado.
  6. No GitHub, edite o arquivo daily.json do repositório e substitua o conteúdo pelo novo.

1 Carregar o daily.json atual

Este é o banco de dados consolidado da estação. Você precisa dele para que os dados novos sejam somados aos antigos sem perder nada.

4 Enviar para o GitHub

Depois de baixar o novo daily.json:

  1. Acesse github.com/ernandes-sobreira/estacao-clima-unemat/blob/main/daily.json
  2. Clique no ícone de lápis (Edit this file) no canto superior direito do arquivo.
  3. Apague todo o conteúdo atual (Ctrl+A, depois Delete).
  4. Cole o conteúdo do novo daily.json (Ctrl+V).
  5. Role até o final da página e clique em Commit changes.
  6. Aguarde uns 2 minutos e acesse a plataforma - os dados novos aparecerão automaticamente.
